# Stock Ticker

A real-time stock market data streaming platform built with Java 21 and Spring Boot.

## Tech Stack

- Java 21
- Spring Boot
- Apache Kafka
- Redis
- PostgreSQL
- WebSockets
- Docker
- Kubernetes
- Prometheus
- Grafana

## Architecture

Market Data Producer -> Kafka -> Analytics Service -> WebSocket Gateway -> Clients

## Status

Project under active development.
